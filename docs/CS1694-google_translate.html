概述 Overview
CS1694 是一种动态LED 控制/驱动器具有多种显示模式6×11 或7×10 The CS1694 is a dynamic LED control/driver with multiple display modes 6×11 or 7×10
内部具有8 级亮度 8 levels of brightness inside
控制带有按键扫描功能可以扫描10×2 的键盘矩阵电路内部含有一个14×8 的显示RAM 以及 Control with a key scan function to scan a 10×2 keyboard matrix circuit containing a 14×8 display RAM and
一个5×6 的按键数据RAM 作为MCU 的理想外围元件带有可以直接和MCU 相接的串行口Din A 5 × 6 button data RAM as an ideal peripheral component of the MCU with a serial port Din that can be directly connected to the MCU
& Dout Stb Clk & Dout Stb Clk
CS1694 是在CS1684 的基础上封装改动形成的在CS1684 电路基础上将Din 与Dout 键合在 The CS1694 is based on the CS1684 package and is based on the CS1684 circuit. Din and Dout are bonded together.
一起Key3 和Seg11 不键合其他功能与CS1684 相同 Key3 and Seg11 are not bonded together and other functions are the same as CS1684
该电路主要用作动态LED 控制/驱动器也有按键扫描的用途是单片机与动态LED 屏之间的 This circuit is mainly used as a dynamic LED control / driver and also has the purpose of button scanning between the microcontroller and the dynamic LED screen.
理想元器件可以用在DVD VCD 功放等整机上 Ideal components can be used on DVD VCD amplifiers

功能说明 Function Description
电路上电后RC 振荡器起振然后通过内部的复位电路对电路进行初始化如操作模式显示 After the circuit is powered up, the RC oscillator starts to oscillate and then initializes the circuit through the internal reset circuit.
模式等电路初始化后可通过串行接口从MCU 接受指令数据等信息只有当Stb 为低电平时 After the circuit such as mode is initialized, information such as command data can be received from the MCU through the serial interface only when Stb is low.
Clk 才有效才可以通过Din & Dout 端口与MCU 之间交换信息接收数据时Stb 从高电平变为低 Clk is valid only when the data is received by the Din & Dout port and the MCU. When the data is received, Stb changes from high to low.
电平后的第一个byte 作为指令用无论是接受数据还是发送数据总是从字节的最低位开始 The first byte after the level is used as an instruction, whether it is accepting data or sending data, always starting from the lowest byte of the byte.
从数据流程看电路内含有三条数据流 From the data flow, there are three data streams in the circuit.
1 串行口串/并转换数据RAM Seg 输出 1 serial port serial/parallel conversion data RAM Seg output
2 串行口串/并转换指令译码控制信息 2 serial port serial/parallel conversion instruction decoding control information
3 按键口按键锁存按键RAM 串并转换Dout 输出 3 button port button latch button RAM string and convert Dout output

示控制指令指令的高两位是指令识别码用以识别所接受的是何种指令然后指向对应的译码器 The upper two bits of the control instruction instruction are the instruction identification code for identifying which instruction is accepted and then pointing to the corresponding decoder.
产生对应的微操作每条指令里的不相关位可以为1 也可以为0 对电路的动作不影响 Generate the corresponding micro-operation. The irrelevant bit in each instruction can be 1 or 0. It does not affect the action of the circuit.
